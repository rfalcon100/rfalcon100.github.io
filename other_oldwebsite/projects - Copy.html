(Untitled) - Custom Ear Trainer

Ear training is a vital skill for both musicians and audio engineers. While there are numerous tools available that help musicians develop a critical hearing, there a very few tools that target audio engineers. I am developing a simple program that employs the techniques I've used to train my hearing.

Key Features:

  • Compose a reference track by importing audio files to the application.
  • Apply random EQ to the reference track within custom parameters.
  • Apply random dynamics variance to the reference track within custom parameters.
  • Keep track of the training with detailed statistics.

Bachelor Thesis: Analysis of short sounds replication using primitive waveforms synthesis and genetic algorithms

For my Bachelor thesis I built an application to study audio and genetic algorithms. The idea behind the project was to use additive synthesis to replicate short sounds, using a combination of sine waves (and other waveforms). The concept was inspired by Roger Johansson's project that replicates images using a combination of simple polygons obtained with genetic algorithms.

Key Features:

  • Evolution parameters highly customizable.
  • Presents comprehensive information about the evolution process.
  • Applies a standard ADSR envelope.
  • Available waveforms include sine, triangular, square and sawtooth.

Custom Files Backup Application

Afraid of data loss, I built and application to do custom backups. Besides copying specified files to different drives, the program also keeps a detailed log of all movements, transactions and metadata for some filetypes. The metadata is particularly useful for music files, as it allows explore my own tastes and learn, for example, the most popular release years from all the music I listen to.
