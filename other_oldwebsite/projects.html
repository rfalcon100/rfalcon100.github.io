Image texture synthesis with Deep Neural Networks

Term project for the Deep Learning course in Aalto University. We implemented a method for high resolution texture synthesis based on Texture Synthesis Using Convolutional Neural Networks by Gatis et al and High Resolution Multi-Scale Neural Texture Synthesis by Snelgrove.

The method uses a deep neural network to extract statistical features of some input image, and then uses the same network to update the pixel values of a generated image by minizing the loss function between the features of the input and generated images. High resolution support is added using a Gaussian pyramid.

Key Features:

  • Brief literature review for texture synthesis and style transfer using deep neural networks.
  • Developed in Python and Tensorflow.

  • Results on images that are not strictly textures.

Parametric Audio Equalization using Genetic Algorithms

Equalizing (filtering) audio signals can be difficult for non tehcnical people as it requires understanding of complex tools such as equalizers and their parameters. Additionally, with some tools like the parametric equalizers it is not possible to obtain any arbritary equalization curve due to the nature of the controls.

I implemented a small Matlab application roughly based on "Graphical Control of a parametric equalizer" by Loviscach. It lets the user draw any arbitray target equalization curve, and then the parameters for a 5 band parametric equalizer are automatically selected via a genetic algorithm so that the equalizer matches the target curve as close as possible.

Key Features:

  • Multiple visualizaiton options to analyze the evolution process.

(Untitled) - Custom Ear Trainer

Ear training is a vital skill for both musicians and audio engineers. While there are numerous tools available that help musicians develop a critical hearing, there are very few tools that target audio engineers. I am developing a simple program that employs the techniques I've used to train my hearing.

Key Features:

  • Compose a reference track by importing audio files to the application or use default tracks like pink noise.
  • Apply random EQ to the reference track within custom parameters.
  • Apply random dynamics variance to the reference track within custom parameters.
  • Keep track of the training with detailed statistics.

Bachelor Thesis: Analysis of short sounds replication using primitive waveforms synthesis and genetic algorithms

For my Bachelor thesis I built an application to study audio and genetic algorithms. The idea behind the project was to use additive synthesis to replicate short sounds, using a combination of sine waves (and other waveforms). The concept was inspired by Roger Johansson's project that replicates images using a combination of simple polygons obtained with genetic algorithms. The full text (in spanish) is available here, and in the University website.

Key Features:

  • Evolution parameters highly customizable.
  • Presents comprehensive information about the evolution process.
  • Applies a standard ADSR envelope.
  • Available waveforms include sine, triangular, square and sawtooth.

Custom Files Backup Application

Afraid of data loss, I built and application to do custom backups. Besides copying specified files to different drives, the program also keeps a detailed log of all movements, transactions and metadata for some filetypes. The metadata is particularly useful for music files, as it allows explore my own tastes and learn, for example, the most popular release years from all the music I listen to.
